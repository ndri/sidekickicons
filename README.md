# Sidekickicons

Fanmade icons meant to complement [Heroicons](https://github.com/tailwindlabs/heroicons).

From the Heroicons README:

> We're not accepting contributions for new icons [...]. Instead we encourage you to release your own icons in your own library [...].

I like to use Heroicons, but it doesn't have some icons I need. Instead of using another icon library, I just create those icons as a sidekick to Heroicons that share a similar style and size. This repository is a collection of those icons.
